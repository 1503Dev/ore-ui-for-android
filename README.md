# OreUI for Android
安卓Minecraft OreUI组件库  
Android OreUI Component Library

## 介绍
纯 Kotlin 继承 View 和 AppCompatView 实现，基本兼容 Java，无 xml 等其他资源  
根据 OreUI 标准和实际游戏效果实现  
使用令人忍俊不禁的 StyleSheet 类管理样式  
使用令人忍俊不禁的 Pixels2D 类管理像素图  
Something in progress...

## 使用
Demo和aar/jar在这里: [Release](https://github.com/1503Dev/ore-ui-for-android/releases)

先导入库，由于不依赖资源，jar或者aar都行  
需要导入AppCompat库，否则会无法运行

- [Kotlin示例](./app/src/main/java/dev1503/oreuiforandroid/FirstFragment.kt)
- [Java示例](./app/src/main/java/dev1503/oreuiforandroid/JavaDemoActivity.java)
